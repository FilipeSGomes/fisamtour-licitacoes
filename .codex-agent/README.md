# .codex-agent

Camada de governança de IA criada para este projeto.

## Primeira execução

```bash
./.codex-agent/scripts/start-discovery.sh
```

## Criar contrato de mudança

```bash
./.codex-agent/scripts/new-change.sh "descrição curta da mudança"
```

## Princípio

Primeiro entender.  
Depois documentar.  
Depois dividir.  
Depois criar prompts.  
Só então alterar.
